## git status 확인 포인트
- 수정 전: `nothing to commit, working tree clean`
- 파일 수정 후: `modified`와 `untracked files`가 표시됨
- `git add` 후: `Changes to be committed`로 변경됨
- 커밋 후: 다시 `working tree clean`으로 변경됨
